# PWA 安裝說明

## 📱 如何安裝「浪LIVE 輪盤紀錄」PWA

### 桌面瀏覽器（Chrome / Edge）

1. **開啟應用程式**
   - 在 Chrome 或 Edge 瀏覽器中開啟應用程式網址
   - 確保使用 HTTPS 或 localhost（開發環境）

2. **查看安裝提示**
   - 瀏覽器地址列右側會出現「安裝」圖示（⊕ 或 +）
   - 或點擊選單（三個點）→「安裝應用程式」

3. **點擊安裝**
   - 點擊「安裝」按鈕
   - 確認安裝對話框

4. **完成安裝**
   - 應用程式會出現在：
     - Windows: 開始選單和桌面
     - macOS: 應用程式資料夾和 Dock
     - Linux: 應用程式選單

### 手機瀏覽器（Android - Chrome）

1. **開啟應用程式**
   - 在 Chrome 瀏覽器中開啟應用程式網址

2. **安裝提示**
   - 瀏覽器會自動顯示「安裝應用程式」橫幅
   - 或點擊選單（三個點）→「安裝應用程式」或「新增至主畫面」

3. **確認安裝**
   - 點擊「安裝」
   - 應用程式圖示會出現在主畫面上

### 手機瀏覽器（iOS - Safari）

1. **開啟應用程式**
   - 在 Safari 瀏覽器中開啟應用程式網址

2. **新增至主畫面**
   - 點擊底部的「分享」按鈕（方框與向上箭頭）
   - 選擇「加入主畫面」
   - 確認名稱和圖示
   - 點擊「加入」

3. **完成**
   - 應用程式圖示會出現在主畫面上
   - 點擊圖示即可開啟（會以獨立視窗模式運行）

## ⚠️ 安裝要求

### 必須條件
- ✅ 應用程式必須在 **HTTPS** 環境下運行（或 localhost 開發環境）
- ✅ 必須有有效的 `manifest.json` 檔案
- ✅ 必須註冊 Service Worker
- ✅ 必須有至少一個圖示（192x192 或更大）

### 檢查清單
- [ ] `manifest.json` 已正確設定
- [ ] `index.html` 中有 `<link rel="manifest" href="./manifest.json" />`
- [ ] Service Worker 已註冊（VitePWA 會自動處理）
- [ ] 應用程式在 HTTPS 或 localhost 上運行

## 🔍 如何檢查是否可安裝

### Chrome DevTools
1. 開啟開發者工具（F12）
2. 切換到「Application」分頁
3. 左側選單選擇「Manifest」
4. 檢查是否有錯誤
5. 查看「Add to homescreen」測試結果

### 檢查項目
- Manifest 是否可讀取
- 圖示是否載入成功
- Service Worker 是否已註冊
- 是否滿足安裝條件

## 🚀 開發環境測試

### 本地測試（localhost）
```bash
npm run dev
# 或
npm run build
npm run preview
```

在 localhost 上也可以安裝 PWA，方便開發測試。

### 生產環境部署
確保部署到支援 HTTPS 的服務（如 Vercel、Netlify 等），才能讓用戶安裝。

## 📝 注意事項

1. **首次安裝**
   - 用戶需要手動觸發安裝（瀏覽器會顯示提示）
   - 不是所有瀏覽器都支援 PWA 安裝

2. **更新應用程式**
   - Service Worker 會自動檢查更新
   - 用戶重新開啟應用程式時會自動更新

3. **離線功能**
   - 安裝後的應用程式可以離線使用
   - Service Worker 會快取必要的資源

## 🐛 無法安裝的常見問題

1. **沒有看到安裝按鈕**
   - 檢查是否在 HTTPS 環境
   - 確認 manifest.json 格式正確
   - 檢查 Service Worker 是否已註冊

2. **圖示顯示異常**
   - 確認圖示路徑正確
   - 檢查圖示尺寸是否符合要求

3. **安裝後無法開啟**
   - 檢查 `start_url` 是否正確
   - 確認應用程式可以正常載入
